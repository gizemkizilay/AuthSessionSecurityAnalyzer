# Auth Session Security Analyzer — Kaynaklar (Seeds)

Bu dosya, araştırma boyunca kullanılacak **birincil standartlar** ve **OWASP rehberleri** için başlangıç kaynak listesidir.

## Standartlar / Spesifikasyonlar
- RFC 6265 — HTTP State Management Mechanism (Cookies): https://datatracker.ietf.org/doc/html/rfc6265
- RFC 6749 — The OAuth 2.0 Authorization Framework: https://datatracker.ietf.org/doc/html/rfc6749
- OpenID Connect Core 1.0: https://openid.net/specs/openid-connect-core-1_0.html
- RFC 8725 — JSON Web Token Best Current Practices (JWT BCP): https://datatracker.ietf.org/doc/html/rfc8725

## OWASP
- OWASP Session Management Cheat Sheet: https://cheatsheetseries.owasp.org/cheatsheets/Session_Management_Cheat_Sheet.html
- OWASP Application Security Verification Standard (ASVS): https://owasp.org/www-project-application-security-verification-standard/
- OWASP Top 10: https://owasp.org/Top10/

## Notlar
- Araştırma sırasında eklenen her yeni kaynağı, ilgili bölüm başlığı altında listele.
- Blog/ikincil kaynakları “İkincil Kaynaklar” bölümüne ekle ve standartlardan ayrı tut.
